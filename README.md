# 當鋪排班表

這是一個可部署到 Vercel 的動態前端排班工具。資料目前儲存在瀏覽器 localStorage，不需要資料庫。

## 功能

- 可新增人員，並指定 A組、B組、D組
- 人員可依組別查看、啟用、停用、刪除
- 產生目前查看週的週一到週六班表
- 已有班表時，重新產生前會先確認
- 所有人模式會重新產生完整班表
- A組、B組、D組模式可只重新產生該組班表，其他組保留不動
- 每次重新產生都會重新隨機該組人員排序
- 班表以一週六天表格呈現，盡量一次顯示完整週班表，不使用固定寬度左右滑動版面
- 所有人模式顯示早班、晚班、中班全部時段
- 單組模式只顯示該組本週輪到的班別時段
- 點擊班表格子中的人員，可直接更換該時段值班人員
- 同一天同一組同一人重複值班時，顯示黃色圓點提示
- 支援整組輪調，例如 A → B → D 改成 B → D → A
- 後續週次會依整組輪調基準自動往下一班輪替
- 可複製目前畫面班表文字
- 可下載目前畫面 CSV
- 每次開啟網頁會自動清除已結束週次班表；網頁開著時每分鐘檢查一次

## 班別

輪班順序：早班 → 晚班 → 中班

- 早班：09:00～13:00
- 晚班：17:00～21:00
- 中班：13:00～17:00

## 組別輪調

預設基準：

- A組：早班
- B組：晚班
- D組：中班

按「整組輪調一次」後：

- B組：早班
- D組：晚班
- A組：中班

再按一次：

- D組：早班
- A組：晚班
- B組：中班

之後每週會依目前基準順序繼續往下一班輪替。

## 狀態機

- Idle：等待操作
- ManagingPeople：新增、停用、啟用、刪除人員
- GeneratingSchedule：產生或重新產生班表
- ViewingSchedule：顯示班表、點擊格子換人
- Error：顯示錯誤訊息

## 部署到 Vercel

1. 到 Vercel 新增專案。
2. 匯入此 GitHub repository。
3. Framework Preset 選 Other。
4. Build Command 可留空。
5. Output Directory 可留空。
6. Deploy。

## 檔案

- `index.html`：頁面結構
- `styles.css`：介面樣式
- `app.js`：狀態機、排班演算法與資料儲存
- `shift-order.js`：班別順序覆寫，將輪班順序設定為早班、晚班、中班
- `vercel.json`：Vercel 路由設定
- `package.json`：專案資訊
