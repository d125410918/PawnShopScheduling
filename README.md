# 當鋪排班表

這是一個可部署到 Vercel 的動態前端排班工具。資料目前儲存在瀏覽器 localStorage，不需要資料庫。

## 功能

- 獨立新增人員
- 人員可指定 A、B、D 三組
- 人員可停用、啟用、刪除
- 從下週一開始產生週一到週六完整排班
- 每人一小時
- 早班、中班、晚班各 4 小時
- 每個時段只會從該組人員中抽選
- 最新加入前 3 名人員有較高排班權重
- 組內人員足夠時，同一天盡量不重複
- 組別內人員不會換組，只會整組輪替到下一個班別
- 可查看所有人，或獨立查看 A組、B組、D組
- 獨立查看組別時，只顯示該組本週輪到的時段
- 可建立範例人員
- 可複製目前畫面排班文字
- 可下載目前畫面 CSV
- 可清除本週排班或全部資料

## 班別

- 早班：09:00～13:00
- 中班：13:00～17:00
- 晚班：17:00～21:00

## 組別輪替

第一週從下週開始：

- A組：早班
- B組：中班
- D組：晚班

下一週自動往下一班輪替：

- A組：中班
- B組：晚班
- D組：早班

再下一週：

- A組：晚班
- B組：早班
- D組：中班

之後每三週循環一次。

## 狀態機

系統使用簡單狀態機控制流程。

- Idle：等待操作
- ManagingPeople：新增、停用、啟用、刪除人員
- GeneratingSchedule：產生排班
- ViewingSchedule：顯示排班
- Error：顯示錯誤訊息

## 部署到 Vercel

1. 到 Vercel 新增專案。
2. 匯入此 GitHub repository。
3. Framework Preset 選 Other。
4. Build Command 可留空。
5. Output Directory 可留空。
6. Deploy。

## 檔案

- `index.html`：頁面結構
- `styles.css`：介面樣式
- `app.js`：狀態機、排班演算法與資料儲存
- `vercel.json`：Vercel 路由設定
- `package.json`：專案資訊
